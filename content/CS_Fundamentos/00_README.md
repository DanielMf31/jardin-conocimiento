## Convenciones

- Cada doc nuevo: **lectura completa** primero, **active recall** 2-3 días después, **atomización** opcional en `10_Notas/` con tag `cs/<tema>`.
- Cuando un Build Things requiera teoría → enlazar al doc relevante de aquí.
- Cuando un doc de aquí necesite ejemplo práctico → enlazar al Build Things relevante.
