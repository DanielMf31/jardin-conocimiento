# 🌱 Jardín de Conocimiento

Apuntes de aprendizaje abiertos sobre **Linux, Hyprland, desarrollo de software, fundamentos de
informática y ciberseguridad**. Pensados para *entender* (esquema y porqué antes que sintaxis) y
conectados entre sí. Empezaron como una bóveda personal de [Obsidian](https://obsidian.md) y se
publican para que cualquiera que tenga interés pueda leerlos y aprender.

## Cómo leerlo

**Opción A — En el navegador (web):**
Publicado con [Quartz](https://quartz.jzhao.xyz) en GitHub Pages. Visita la URL del sitio (ver
"Settings → Pages" del repo) y navega por los enlaces, la búsqueda y el grafo.

**Opción B — En tu portátil, clonando (recomendado para la experiencia completa):**
```bash
git clone <URL-DE-ESTE-REPO>.git
```
Abre la carpeta `content/` como **vault** en Obsidian (gratis). Tendrás wikilinks, grafo,
backlinks y búsqueda nativos. También puedes leer los `.md` con cualquier editor o visor de Markdown.

## Avisos

- **Sin secretos**: se han eliminado contraseñas, datos personales y credenciales antes de publicar.
  Los wargames se incluyen **sin las contraseñas reales** (las tablas de progreso están en blanco
  para que las rellenes tú).
- **Ética**: el material de seguridad es para **defender** y para pentesting **autorizado** (tus
  sistemas, labs, CTFs). Los walkthroughs de wargames son educativos; intenta cada nivel tú primero.
- **Calidad**: son apuntes de estudio en mejora continua, hechos con apoyo de IA y revisión propia.
  Pueden contener errores; las correcciones (issues / PRs) son bienvenidas.
